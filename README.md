# Jesus Festival Movement

> From Hamilton, Ontario to the nations — raising up evangelistic festivals that become lasting Gospel movements.

A cinematic, SEO-optimized website for the **Jesus Festival Movement**, a global evangelistic movement that began in Hamilton, Ontario, Canada.

**Gather. Worship. Preach Jesus. Reach the lost. Disciple believers. Plant ongoing outreach. Multiply movements around the world.**

## Tech Stack

- **[Next.js 14](https://nextjs.org/)** (App Router)
- **[Tailwind CSS](https://tailwindcss.com/)**
- **[Framer Motion](https://www.framer.com/motion/)** for scroll-triggered reveals & cinematic motion
- Dynamic Open Graph image (`next/og`)
- JSON-LD schema (Organization, WebSite, Event) for rich SEO

## Local Development

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
```

## Deployment

Optimized for [Vercel](https://vercel.com/). Push to the connected repo or run `vercel --prod`.

> Update `SITE.url` in [`lib/content.ts`](lib/content.ts) to your production domain so canonical URLs, sitemap, and Open Graph metadata resolve correctly.
